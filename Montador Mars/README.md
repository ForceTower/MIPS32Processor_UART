# MARS (Montador e Simulador MIPS)

## Uma IDE para Programação de Linguagem de Montagem MIPS

MARS é um ambiente leve de desenvolvimento interativo (IDE) para programação em linguagem de montagem MIPS, destinado ao uso em nível educacional pela Patterson e Hennessy Computer Organization and Design.
